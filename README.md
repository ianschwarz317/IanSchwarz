# ianschwarz.com

Personal site for Ian Schwarz — builder of AI-powered products across pet health,
hospitality, payments, and productivity.

Single static page (`index.html`, self-contained HTML/CSS/JS) deployed via GitHub
Pages on the custom domain in `CNAME` (ianschwarz.com).

## Products featured
- **Honeybee** — free pet microchip registry & health-records platform — https://seehoneybee.com/
- **DirectGuest** — AI-native hotel distribution channel — https://directguest.ai/
- **BlackoutPDF** — in-browser PDF redaction — https://blackoutpdf.co/
- **SureHold** — lien-waiver payment automation for GCs — https://surehold.xyz/
- **Raiz'd** — fundraising operations platform — https://raizd.xyz/
- **Wallio** — digital business cards in Apple/Google Wallet — https://wallio.co
- **FeedAnchor** — LinkedIn post saver (Chrome extension) — https://feedanchor.com

## Editing
Everything lives in `index.html`. `robots.txt` and `sitemap.xml` must keep those
exact filenames so crawlers find them at the site root.
